## Context

## 핵심 설계

### 1. 문제집 번호 포맷

```
E2-03-X7K2M
```

| 파트 | 예시 | 의미 |
|---|---|---|
| 학년 코드 | `E1` `E2` `E3` `M1` | E1=초1-2, E2=초3-4, E3=초5-6, M1=중학교 |
| 챕터 번호 | `03` | 해당 학년의 3번째 챕터 (01~99, 확장 가능) |
| 시드 | `X7K2M` | 5자리 Base36 대문자 난수 (랜덤) |

#### URL 구조

```
https://alsudawa.github.io/math/?wid=E2-03-X7K2M
```

파라미터가 `wid` 하나. 로드 시:
- `wid` 없으면 → 기본 학년+챕터에서 새 시드 생성 후 `history.replaceState`
- `wid` 있으면 → 파싱 → 동일 문제 재현

### 5. 시드 기반 LCG 난수 생성기

```js
class SeededRandom {
  constructor(seedStr) {            // seedStr: 'X7K2M' 같은 문자열
    this.state = parseInt(seedStr, 36) ^ 0x12345678;
  }
  next() {
    this.state = Math.imul(1664525, this.state) + 1013904223 | 0;
    return (this.state >>> 0) / 0x100000000;
  }
  int(min, max) { return min + Math.floor(this.next() * (max - min + 1)); }
}

function newSeed() {
  return Math.floor(Math.random() * 60466175)   // 36^5 - 1
    .toString(36).toUpperCase().padStart(5, '0');
}
```

#### 장 수 입력 → 인쇄 동작

- **장 수 입력**: 1~20 (숫자 입력 또는 ▲▼ 버튼)
- **생성**: N장 × 각기 다른 랜덤 시드 생성 → 화면에 1장씩 미리보기 (◀ ▶로 탐색)
- **인쇄**: `window.print()` → A4 N페이지 문제지 + N페이지 답안지 (총 2N 페이지)

```
인쇄 출력 순서:
  [문제지 1장] [문제지 2장] ... [문제지 N장]
  [답안지 1장] [답안지 2장] ... [답안지 N장]
```

각 인쇄 페이지에 **해당 장의 고유 번호 + QR코드** 인쇄.

#### URL 처리

- N장 생성 시 URL은 **1번 장의 wid**만 반영 (`?wid=E2-03-X7K2M&n=5`)
- `n` 파라미터로 연속 시드 재현 가능: seed+1, seed+2, ... 방식

```js
// 1번 장 시드를 기준으로 나머지 시드를 결정적으로 파생
function deriveSeeds(baseSeed, count) {
  const rng = new SeededRandom(baseSeed);
  return Array.from({length: count}, () =>
    Math.floor(rng.next() * 60466176).toString(36).toUpperCase().padStart(5,'0')
  );
}
```

## 구현 단계

1. HTML 골격 + 컨트롤 패널 (학년 탭 → 챕터 라디오 버튼 + 장 수 입력)
2. `SeededRandom` 클래스 + `newSeed()` + `deriveSeeds(base, n)` 함수
3. `wid` 파싱/생성 로직 + `n` 파라미터 처리
4. 챕터별 문제 generator 함수 구현 (6그룹 39챕터, 각 `perPage` 포함)
5. N장 생성 + 화면 미리보기 페이지네이션 (◀ 1/N ▶)
6. 문제 + 답안 DOM 렌더링
7. 컬러풀한 화면 CSS + A4 최적화 `@media print`
   - 인쇄 시 N장 문제지 → N장 답안지 순서로 출력
   - 각 장마다 `page-break-after: always`
8. QR코드 렌더링 (현재 미리보기 장 기준)
9. 답안 토글 + 인쇄 버튼 + 문제집 번호 직접 입력 동작
10. git commit & push → `claude/math-problem-generator-5JmYS`
